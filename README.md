# TunnelTool
TunnelTool is a small tool helping at accessing servers

  - Open SSH Tunnels to various servers
  - Download Logfiles
  - Open Database Connections
  - Upload Files to Servers
  
## Build and Run
Checkout the project and use
```sh
$ mvn package
```
